# 「波束の収縮」から「量子インストルメント」へ：測定概念の拡張

学部レベルの量子力学で学ぶ「測定すると固有状態へ一瞬で遷移する」という定義（波束の収縮）は、量子情報理論においてより柔軟で一般的な「量子インストルメント」へと拡張されています。そのステップを整理します。

---

## 1. 第1段階：射影測定 (PVM)
**「これまでの常識：測定＝固有状態への収縮」**

学部レベルの量子力学（射影測定：Projective Measurement）では、測定は以下のように定義されます。

*   **ルール**: 観測量 $H$ の固有状態 $\{|a\rangle\}$ への射影演算子 $P_a = |a\rangle\langle a|$ を用いる。
*   **事後状態**: 測定結果 $a$ を得た直後の状態 $\rho'$ は、必ずその固有状態になります。
    $$ \rho \xrightarrow{\text{結果 } a} \rho' = \frac{P_a \rho P_a}{\operatorname{Tr}(P_a \rho)} = |a\rangle\langle a| $$

ここでは、**「何を得るか（確率）」と「何になるか（事後状態）」が、同じ演算子 $P_a$ によって一意に決まってしまっています。**

---

## 2. 第2段階：POVM (Positive Operator-Valued Measure)
**「確率の一般化：何を得るか、だけに集中」**

量子情報の初期段階では、測定を「統計的なサンプリング」として捉え直し、POVM を導入します。

*   **ルール**: $M_a \ge 0$ かつ $\sum M_a = \mathbb{I}$ を満たす演算子の集合。
*   **確率**: $p(a) = \operatorname{Tr}(M_a \rho)$
*   **事後状態**: **規定されない。**

POVM は「どのような確率でデータが得られるか」を記述するのには完璧ですが、**「測定した後に粒子がどうなったか」には関心がありません。** 例えば、光子を検出器で吸い込んで消滅させてしまうような測定（破壊的測定）も POVM で記述できますが、その後に「固有状態 $|a\rangle$ になる」と言うのは物理的に不自然です。

---

## 3. 第3段階：量子インストルメント (Quantum Instruments)
**「完成形：確率と事後状態をバラバラに定義する」**

現実の測定（ノイズがある、不完全、あるいは測定後も系が残る）を記述するために導入されたのが「量子インストルメント」です。

*   **ルール**: 測定結果 $a$ ごとに、**完全正値 (CP) 写像** $\mathcal{A}_a$ を割り当てる。
*   **和の条件**: 全ての写像の和がチャネル（トレース保存写像） $\sum \mathcal{A}_a = \Phi$ になる。
*   **確率**: $p(a) = \operatorname{Tr}(\mathcal{A}_a(\rho))$
*   **事後状態**: 
    $$ \rho \xrightarrow{\text{結果 } a} \rho' = \frac{\mathcal{A}_a(\rho)}{\operatorname{Tr}(\mathcal{A}_a(\rho))} $$

### ここが拡張のポイント！
1.  **「固有状態」に縛られない**: 
    写像 $\mathcal{A}_a$ は自由度が高いため、測定後に状態が少しだけ歪む、別の基底に写る、あるいは補助系と絡み合う、といった複雑なプロセスをすべて記述できます。
2.  **Kraus 演算子による記述**:
    各 $\mathcal{A}_a$ を Kraus 表現すると $\mathcal{A}_a(\rho) = \sum_j A_{a,j} \rho A_{a,j}^\dagger$ と書けます。学部レベルの射影測定は、Kraus 演算子が単一の射影演算子 $A_{a,1} = P_a$ である特殊なケースに過ぎません。
3.  **POVM との接続**:
    $$ M_a = \sum_j A_{a,j}^\dagger A_{a,j} $$
    と定義すれば、インストルメントから自動的に POVM が導かれます。

### 3.3 数式と物理プロセスの対応

量子計算における「測定」という動的なプロセスと、インストルメントの数式は以下のように対応しています。

| 物理的プロセス | 対応する数式・要素 | 解説 |
| :--- | :--- | :--- |
| **1. 測定器と系の相互作用** | Kraus 演算子 $A_{a,j}$ | 系が測定器（補助系）とユニタリ相互作用した結果、系に残された「副作用」を反映しています。 |
| **2. 情報の古典的な読出し** | 指標 $a$ と確率 $p(a)$ | 相互作用の結果、どのアウトカム $a$ が確定したかを表します。確率は $p(a) = \operatorname{Tr}(\mathcal{A}_a(\rho))$。 |
| **3. それに応じた「変形」** | 写像 $\mathcal{A}_a$ | これまでの「射影」を一般化した部分です。測定結果 $a$ に基づき、系に加わるあらゆる状態変化（回転、ノイズ、破壊）を記述します。 |

数式 $\mathcal{A}_a(\rho)$ は、単に確率を与えるだけでなく、**「情報を抜いたあとの、系への物理的な副作用」**をすべて一括して引き受けている演算子だと言えます。

---

## まとめ：何が変わったのか？

| 特徴 | 学部レベル (PVM) | 量子情報 (Instrument) |
| :--- | :--- | :--- |
| **遷移先** | 必ず測定の固有状態 | 任意の（写像による）状態 |
| **演算子の制約** | べき等かつ直交 ($P_a^2 = P_a$) | 任意の CP 写像（和がトレース保存） |
| **物理的イメージ** | 波束の完全な収縮 | 測定器との相互作用による状態変化 |

結論として、量子情報では「測定＝固有状態への遷移」という強い制約を外し、**「測定器と系が相互作用し、その一部の情報が古典的に読み取られ、それに応じた『なんらかの変形』が系に加わる」**という、より物理的で広範なプロセスとして測定を再定義しているのです。
