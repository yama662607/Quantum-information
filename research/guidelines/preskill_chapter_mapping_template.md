## 記入のコツ

### ページ開始位置の決定方法

以下の順で各セクション開始ページを特定：

1. **PDF を開く** → 物理的なページスクロール（画面上の「ページX」）
2. **セクション見出しを探す**（例: 「1.1 Quantum States」）
3. **そのセクションが始まったページ番号を記録**
4. **複数ページ確認** → 次のセクション開始ページ = 前セクションの終了ページ

### 内容のポイント欄

各セクションに **含まれる重要な要素** を簡潔に記録：

```
例:
- 定義 2.1: 密度行列の定義
- 定理 2.3: トレース条件の性質
- 図 2.2: ブロッホ球の例
- Box 2.1: 補足（オブザーバブル）
```

### 備考欄

- 図表が多い箇所 → 「図多数」
- 複雑な数式 → 「複雑な証明」
- 参考文献への言及 → 「参考文献: p.XX」

## Quarto での使用方法

このマッピング情報を基に、`quarto/textbook-preskill/chapter1/_*.qmd` ファイルに PDFリンクを記入：

```markdown
## 1.1 量子状態とアンサンブル [chap1.pdf](/assets/pdf/preskill/chap1.pdf#page=5) {#sec-quantum-states}

（本文内容）
```

**ルール**：
- `#page=N` には、このマッピング表の「開始ページ」を記入
- 一度記入したら、変更しないこと（後の参照が破壊される）

## よくある質問

### Q: ページ番号が PDF ビューアと異なる

**A**: `just process-pdf` で表示されたページカウント（内部ページ数）と、PDF ビューアの表示（通常ページ）が異なる場合があります。
- → **PDF ビューアの表示に合わせる**（通常 p.1, p.2, ...）
- → マッピング表にも「p.1 = ビューア表示 1」と明記

### Q: セクション見出しがない場合

**A**: セクション見出しが紙面に印字されていない場合、以下の方法で区別：
- 改行や段落の大きさの変化
- フォント・サイズの変化
- 番号振りの変化（例: 前のセクションが 1.2 なら、次は 1.3）

### Q: 複数ページにまたがるセクション

**A**: 「開始ページ」「終了ページ」で範囲を示す：
- セクション 1.2 = p.5-12（全8ページ）
- セクション 1.3 = p.13-20（全8ページ）
